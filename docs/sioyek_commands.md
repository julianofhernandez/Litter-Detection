## Basics

**O** open a new document

**Shift + O** open a previous document

**F10** fit to screen

**T** table of contents

## History

**Backspace** go back in history

**Ctrl + ->** go forwards in history

**Ctrl + <-** go backwards in history

## Search

**Ctrl + F** search

**N** next

**Shift + N** previous

## Highlights

**H -> \<Key\>** add

**D -> H** remove

**G -> H** search

**G -> Shift + H** global search

## Colors
**G**reen - Dataset

**Y**ellow - General

**R**ed - CNN Model
